## Bubble Sort
Time Complexity: O(n^2)

Sort by comparing and swapping adjacent elements

Example) [6, 4, 1, 3]

1st Round

[4, 6, 1, 3] -> [4, 1, 6, 3] -> [4, 1, 3] [6]

2st Round

[4, 1, 3, 6] -> [1, 4, 3, 6] -> [1, 3, 4, 6]

3rd Round 

[1, 3, 4, 6] -> No change

4th Round 

[1, 3, 4, 6] -> No change
